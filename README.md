# Data Science Internship Tasks

## Task 1: Iris Visualization
**Objective:** Visualize the Iris dataset to understand feature distributions.  
**Approach:** Used matplotlib and seaborn for scatterplots and pairplots.  
**Results & Insights:** Species are clearly separated in pairplots.

## Task 2: Credit Risk Prediction
**Objective:** Predict loan default risk using dataset.  
**Approach:** Preprocessed data, trained Random Forest model.  
**Results & Insights:** Accuracy: 85%, feature importance highlights credit history.

## Task 4: Insurance Claim Prediction
**Objective:** Predict insurance claim approval.  
**Approach:** Used logistic regression and decision tree models.  
**Results & Insights:** Decision tree had better recall on claim approvals.
